# Screenshots

Coloca aquí las capturas de pantalla de la aplicación.

## Capturas recomendadas:

1. **dashboard.png** - Vista principal del dashboard
2. **empleados.png** - Panel de gestión de empleados
3. **fichaje.png** - Sistema de fichaje con geolocalización
4. **solicitudes.png** - Gestión de solicitudes (vacaciones/permisos)
5. **informes.png** - Generación de informes PDF
6. **login.png** - Página de inicio de sesión

## Consejos para las capturas:

- Usa resolución 1920x1080 o similar
- Formato PNG para mejor calidad
- Oculta datos sensibles reales si existen
- Muestra ejemplos de datos de prueba
- Captura pantalla completa o recorta el contenido relevante
